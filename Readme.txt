Instructions to run the code:

1.Store the program that is attached as a onedrive link in the project document as NetworkDesign.java file.

2.Open the command prompt

3.Open the location where the program has been stored

4.Create a .class file by using command

javac NetworkDesign.java

5.Once it is created, run the program using the command

java NetworkDesign

6. Input the number of nodes as N=21

7. Input random values for k in the set [0,1,2,3]

8. Enter the values for k (varying values)

9. The program will run and display the cost, density and the matrix representation of network topology.
